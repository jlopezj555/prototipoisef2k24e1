Traslado De Productos

En este formulario podrás gestionar el traslado de productos entre bodegas de manera eficiente y segura. A continuación, se explica cada sección y el uso correcto del formulario.

Guía Técnica Paso a Paso

1. Selecciona las Bodegas
En la parte izquierda, elige la Bodega de Entrada (destino) y la Bodega de Salida (origen) usando los menús desplegables.
Ejemplo: Entrada: Bodega Central, Salida: Bodega Norte.

Nota: No coloques las mismas bodegas en ambos campos, ya que no se permitirá continuar con el traslado.
------------------------------------------------
2. Selecciona el transporte
    Selecciona el vehículo que realizará el traslado.
    El peso del Vehiculo(Peso V). En este campo se mostrará el peso correspondiente al vehículo seleccionado.
    Selecciona la fecha en la que se realizará el traslado(Fch. Traslado).
    ------------------------------------------------
    3. Agrega Productos al Traslado:
    • Selecciona el código del producto.
      El sistema mostrará: El ID, Nombre, Peso, stock, y CostoU disponible del producto.
      Al ingresa la cantidad del producto a trasladar.
      El sistema calculará el Costo Total.

      Al Haser clic en el botón Agregar ().
      los productos se añadirán a la lista de productos a trasladar y se mostrarán en el DataGridView.
    ------------------------------------------------
    4. Detalle De Movimiento Salida/Entrada
      El DataGridView. Mostrará los documentos generados correspondientes a cada traslado.
      El contenido que se podrá visualizar incluye: No. de traslado, documento, fecha, costo total, costo total general, precio total y código del producto.
      Se muestra un apartado llamado Tipo de Informe:, donde se pueden observar dos opciones. En este momento, se visualiza el informe de salida del traslado de productos, el cual se muestra en el DataGridView. Esto se debe a que se generan dos documentos por cada traslado: uno de salida y otro de entrada.
      A continuación se muestra la información correspondiente al traslado de salida.

      Al seleccionar la fila del traslado, se desplegará una ventana en la que se podrá visualizar la lista de
      productos a trasladar, junto con la bodega de origen y la bodega de destino.

      Aquí se ha cambiado el tipo de informe a Informe Entrada TDP. El cual corresponde al segundo
      documento generado. Este documento posee las mismas características que el primero, con la diferencia
      de que se trata de un informe de entrada. En él se puede visualizar de dónde salió el producto y el estado en el
      que se encuentra. Dicho estado cambiará de 'Activo' a 'Cancelado' en caso de que el traslado sea cancelado.

      ------------------------------------------------

      Descripción de los Botones


        1. Agregar: Este botón permite limpiar los campos para ingresar un nuevo traslado.
        2. Reportes: Este botón permite visualizar y acceder a los reportes de los traslados realizados.
        3. Ayuda: Este botón brinda acceso a la información sobre el uso del formulario.
        4. Productos Disponibles En Bodegas: Al hacer clic en este botón, se visualizarán los productos ingresados a la bodega, así como el stock que ha sido añadido con cada ingreso

        5. Cancelar Traslado: Este botón permite anular un traslado y retornar los productos a la bodega desde la que fueron enviados.
        6. Aceptar Traslado: Por último, este botón permite confirmar o aceptar un traslado.
      ------------------------------------------------

      Advertencias y Buenas Prácticas

      • - Verifica siempre el stock disponible antes de ingresar la cantidad a trasladar.
        - El peso total de los productos no debe superar la capacidad del vehículo.
        - Completa todos los campos obligatorios para evitar errores de validación.
        - Utiliza el botón Actualizar para ver los últimos movimientos registrados.
      ------------------------------------------------

      Preguntas Frecuentes

      • ¿Qué hago si el formulario no se envía? Verifica que todos los campos estén completos y que el stock y peso sean correctos.
        ¿Puedo agregar varios productos? Sí, repite el proceso de agregar para cada producto distinto.
        ¿Dónde veo el historial? Usa el botón de reportes para consultar movimientos anteriores.
        ¿Qué significa "Stock Restante"? Es la cantidad de producto que quedará disponible en la bodega de salida después del traslado.
      ------------------------------------------------

      Ejemplo Completo de Uso

        1. Selecciona las "Bodegas" entrada y salida.
        2. Elige el "Vehiculo" y Selecciona la fecha en la que se realizará el traslado "16/05/2024".

        3. Selecciona el codigo del producto ejemplo: "101", En caso de no contar con el stock necesario para el traslado, se mostrará una advertencia, ingresa una cantidad ejemplo:"10" Esta cantidad puede variar dependiendo del número de productos que se desee trasladar.

        4. Haz clic en el botón Agregar. Para seguir añadiendo más productos al traslado.
        5. Verifica la lista, luego haz clic en Aceptar Traslado.
        6. Si todo es correcto, el sistema confirmará el traslado y actualizará los stocks.
