Funciones de los Botones

  • Botón Enviar Solicitud: Inicia el proceso para realizar una solicitud de mantenimiento, capturando todos los datos necesarios.
  • Botón Generar PDF: Abre una ventana para generar un reporte en formato PDF de las solicitudes de mantenimiento.
  • Botón Editar Solicitud: Permite modificar la solicitud de mantenimiento seleccionada en el DataGridView.
  • Botón Eliminar Solicitud: Elimina la solicitud de mantenimiento seleccionada, asegurándose de que se haya hecho una selección válida.
  • Botón Actualizar Datos: Refresca el DataGridView para mostrar la información más reciente de las solicitudes de mantenimiento.
  • Botón Nuevo Registro: Limpia los campos para iniciar un nuevo registro de mantenimiento.
  • Botón Reporte: Abre una nueva ventana para visualizar el reporte de mantenimiento previamente seleccionado.

Preguntas Frecuentes

  • ¿Cómo puedo enviar una solicitud de mantenimiento? Debes llenar todos los campos requeridos y presionar el botón "Enviar Solicitud".
  • ¿Por qué no puedo editar o eliminar una solicitud? Asegúrate de haber seleccionado una solicitud en el DataGridView antes de intentar editar o eliminar.
  • ¿Qué debo hacer si el DataGridView no muestra datos? Verifica que haya solicitudes de mantenimiento en la base de datos y presiona el botón "Actualizar Datos" para refrescar la vista.
  • ¿Cómo genero un reporte en PDF? Presiona el botón "Generar PDF" para abrir la ventana de generación de reportes.
  • ¿Qué hago si hay un error al enviar la solicitud? Revisa los datos ingresados y asegúrate de que todos los campos obligatorios estén completos antes de enviar.
