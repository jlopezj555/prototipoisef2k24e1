Promociones


Función del Formulario:

El formulario de Promociones permite ingresar, registrar promociones a empleados que como tal, crecen en la organización




  • Botones: Boton de ingresar, modificar, guardar, cancelar, eliminar, buscar, ver reporte, ver ayuda y salir.

  • Nuevo: Habilita el boton para ingresar datos.

  • Guardar:Permite guardar los datos.

  • Cancelar: Sirve para cancelar la operación.

  • Eliminar: Elimina el registro.

  • Ayuda: Para ver la correspondiente ayuda.

  • Reporte: Para visualizar el reporte asociado con promociones.

  • Salir: Permite salir.

  • Ingresos:Permite como tal el ingreso de una nueva promoción a traves de un ID, un empleado asociado, al cual se obtendra automaticamente su salario y puesto actual, luego se podra seleccionar el puesto nuevo, y nos dara su salario, para luego registrar un motivo y asi guardar la promoción.




  • Ver datos:En el DataGridView se podrá visualizar todo lo relacionado con los datos de la promoción ingresada, siendo el caso donde podremos ver el ID, empleado, puesto actual, salario actual, nuevo puesto, nuevo salario y el motivo.
