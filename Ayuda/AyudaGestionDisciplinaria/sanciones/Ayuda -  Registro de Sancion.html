Formulario


El formulario cosnta de los botones Nuevo, Guardar, Cancelar, Editar, Eliminar, Buscar, Reporte, Ayuda y Salir. 


Objetivo del formulario


Registrar una sancion que se vinculara a una falta previamente reportada en formulario correspondiente.


Registrar una sancion

Se debe presionar el Boton Nuevo, el cual es el primero de izquierda a derecha en la parte superior.


Esto desbloqueara todos los controles necesarios para registrar una sancion y deben ser rellenados todos para que pueda ser guardada. 



Guardar evidencia

Cuando se termine de completar todos los campos necesarios el Boton Guardar se encargara de Guardar la sancion en la base de datos y mostrando una retroalimentacion visual (Tabla que se encuentra en la parte inferior del formulario) del campo recien guardado unicamente y a la vez mostrando un mensaje de si se guardo exitosamente o de lo contrario si hubo algun error al guardar, o si incluso hace falta algun campo/ condicion para poder hacer el guardado.



Limpiar todos los campos

El boton cancelar limpia todos los campos y restablece el formulario a su estado inicial.


Editar un registro

Para editar un registro primero se debe hacer uso del Boton Buscar o del boton Mostrar Eliminados en caso el registro a editar o recuperar sea uno que haya sido dado como sancion invalida o se haya eliminado logicamente.


Buscar o mostrar eliminados


Esto hara que la tabla del formulario muestre los registros correspondientes y para editar uno debe hacerse clic en el registro que desea modificarse o recuperar en caso sea un registro eliminado, ya que el boton editar funciona como un boton de recuperacion en este caso.

 Luego de hacer las modificaciones correspondientes en el registro deseado el boton Editar hara los cambios en dicho registro y mostrara un mensaje como el siguiente si fue exitosa la modificacion.



Eliminar un registro

De igual forma que editar un registro primero se debe hacer uso del Boton Buscar.

Para que luego la tabla del formulario muestre los registros validos existentes y debe hacerse clic en el registro que desea eliminarse. Una vez seleccionado el registro deseado, se podra visualizar toda la informacion previa a eliminarse y si se esta seguro de eliminarse el boton de eliminar hara un borrado logico, lo que significa que el estado de dicha evidencia pasara a ser "0" y provocando que el boton buscar no muestre dicho registro.



Buscar un registro

El boton buscar tiene como funcion mostrar en la tabla del formulario todos los registros validos (Estado = 1) lo que significa que cualquier registro guardado como evidencia no valida no se mostrara con este boton. 


Una vez que el boton buscar despliegue los registros validos en la tabla, este habilitara las funciones de editar y eliminar.


Generar un reporte

Visualizar documento de Ayuda

Este boton permite visualizar el documento de ayuda. 



Salir del formulario

Para salir del formulario se debe oprimir este boton.



Funciones extras del formulario


Boton para Mostrar eliminados



Su funcion es similar al boton buscar, sin embargo a diferencia del boton buscar, este boton se encarga solo de mostrar registros que tengan estado = 0, lo que quiere decir que muestra registros, que fueron eliminados previamente o guardados como faltas que no ameritan sancion, con el objetivo de poder recuperar registros no validos o eliminados.
