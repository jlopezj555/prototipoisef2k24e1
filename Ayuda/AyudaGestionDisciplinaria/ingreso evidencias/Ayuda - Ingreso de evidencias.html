Formulario


El formulario cosnta de los botones Nuevo, Guardar, Cancelar, Editar, Eliminar, Buscar, Reporte, Ayuda y Salir. 


Objetivo del formulario


Ingresar evidencias que se adjuntaran a una falta previamente reportada en formulario correspondiente.


Ingresar una evidencia

Se debe presionar el Boton Nuevo, el cual es el primero de izquierda a derecha en la parte superior.

Esto desbloqueara todos los controles necesarios para poder adjuntar una evidencia y deben ser rellenados todos para que puedan ser guardados. 


Cuando se remarque el control de "si" en el apartado de "Â¿Se adjunto evidencia valida? el formulario se tornara en verde lo que lo guardara como una evidencia con estado "1" (valida) en caso contrario se remarque "no", el formulario se tornara rojo y se guardar la evidencia con el estado "0" (no valida). lo que provocara que al darle al boton buscar no se muestre dicha evidencia en la tabla ubicada en la parte inferior del formulario.



Guardar evidencia

Cuando se termine de completar todos los campos necesarios el Boton Guardar se encargara de Guardar la evidencia en la base de datos y mostrando una retroalimentacion visual (Tabla que se encuentra en la parte inferior del formulario) del campo recien guardado unicamente y a la vez mostrando un mensaje de si se guardo exitosamente o de lo contrario si hubo algun error al guardar, o si incluso hace falta algun campo/ condicion para poder hacer el guardado.



Limpiar todos los campos

El boton cancelar limpia todos los campos y restablece el formulario a su estado inicial.


Editar un registro

Para editar un registro primero se debe hacer uso del Boton Buscar o del boton Mostrar Eliminados en caso el registro a editar o recuperar sea uno que haya sido dado como evidencia invalida o se haya eliminado logicamente.


Buscar o mostrar eliminados


Esto hara que la tabla del formulario muestre los registros correspondientes y para editar uno debe hacerse clic en el registro que desea modificarse o recuperar en caso sea un registro eliminado, ya que el boton editar funciona como un boton de recuperacion en este caso.

 Luego de hacer las modificaciones correspondientes en el registro deseado el boton Editar hara los cambios en dicho registro y mostrara un mensaje como el siguiente si fue exitosa la modificacion.


O si el caso fuera que se selecciono un registro eliminado previamente o que era invalido se mostrara primero este mensaje para confirmar que se desea recuperar o modificar dicho registro:

Si la respuesta es Si, mostrara el mensaje anterior "evidencia actualizada correctamente".


Eliminar un registro

De igual forma que editar un registro primero se debe hacer uso del Boton Buscar.

Para que luego la tabla del formulario muestre los registros validos existentes y debe hacerse clic en el registro que desea eliminarse. Una vez seleccionado el registro deseado, se podra visualizar toda la informacion previa a eliminarse y si se esta seguro de eliminarse el boton de eliminar hara un borrado logico, lo que significa que el estado de dicha evidencia pasara a ser "0" y provocando que el boton buscar no muestre dicho registro.


En caso se seleccione "Si"



Buscar un registro

El boton buscar tiene como funcion mostrar en la tabla del formulario todos los registros validos (Estado = 1) lo que significa que cualquier registro guardado como evidencia no valida tampoco se mostrara con este boton. 


Una vez que el boton buscar despliegue los registros validos en la tabla, este habilitara las funciones de editar y eliminar.

Visualizar documento de Ayuda

Este boton permite visualizar el documento de ayuda. 



Salir del formulario

Para salir del formulario se debe oprimir este boton.



Funciones extras del formulario 


Boton para Cargar archivo local


Este boton esta ligado al campo de Cargar Archivo, este campo acepta direcciones web (url, ip, etc) y tambien rutas locales, y para el caso de querer cargar la direccion de un archivo local, este boton despliega el explorador de archivos del sistema operativo para poder localizar el archivo que desea registrarse como evidencia, y una vez que se seleccione dicho archivo, el boton automaticamente rellenara el campo con la direccion local de dicho archivo.


Boton para probar URL

Este boton esta ligado al campo de Cargar Archivo, de igual forma que el boton de "Cargar archivo local", sin embargo su funcion es unicamente como apoyo al usuario para verificar si la direccion web proporcionada y colocada en el campo de "Cargar Archivo" es valida sin necesidad de ir al navegador a corroborar si funciona o redirecciona a un sitio valido, sin embargo usar este boton es opcional.

Boton para Mostrar eliminados



Su funcion es similar al boton buscar, sin embargo a diferencia del boton buscar, este boton se encarga solo de mostrar registros que tengan estado = 0, lo que quiere decir que muestra registros, que fueron eliminados previamente o guardados como evidencias no validas, con el objetivo de poder recuperar registros no validos o eliminados.
