Formulario de Evaluación de Empleados

  • Datos del Empleado y Evaluador:
    • Empleado a Evaluar: ComboBox para seleccionar al empleado que recibirá la evaluación.
    • Evaluador: ComboBox para seleccionar al evaluador responsable.
    • Fecha: Selector de fecha para registrar el día de la evaluación.
    • Tipo de Evaluación: ComboBox con opciones:
      • Autoevaluación
      • Evaluación de Compañero
      • Evaluación de subordinado
      • Evaluación de jefe de área
  • Evaluación de Competencias:
    • Lista de competencias con sus respectivos campos:
    • Nota: NumericUpDown para calificar cada competencia (0 a 10).
    • Observaciones: TextBox opcional para comentarios.
  • Evaluación General:
    • Observaciones Generales: TextBox (opcional).
    • Calificación General: TextBox (obligatorio, del 0 al 10).
  • Botones Funcionales:
    • Guardar: Verifica que todos los campos obligatorios estén llenos antes de guardar.
    • Limpiar: Borra todos los campos para una nueva evaluación.
    • Reportes: Abre el formulario de reportes.
    • Ayuda: Muestra este documento.
    • Cerrar: Cierra el formulario actual.

Formulario de Reportes de Evaluación

  • Visualización de Evaluaciones:
    • ComboBox de Empleados: Permite seleccionar un empleado para ver sus evaluaciones.
    • DataGridView: Muestra los resultados de las evaluaciones filtradas.
  • Opciones Adicionales:
    • Buscar: Filtra evaluaciones por empleado seleccionado.
    • Reporte: Genera reporte de Crystal Reports para el empleado seleccionado.
    • Ayuda: Muestra este archivo.
    • Salir: Cierra el formulario.

Preguntas Frecuentes

  • ¿Qué pasa si olvido llenar un campo obligatorio? El sistema no permitirá guardar la evaluación hasta completar todos los campos obligatorios.
  • ¿Puedo modificar una evaluación guardada? No, por seguridad y trazabilidad, las evaluaciones no pueden ser modificadas una vez guardadas.
  • ¿Cómo genero un reporte? Selecciona al empleado en el ComboBox del formulario de reportes y pulsa el botón "Reporte".
